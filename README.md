# uas-nlp-sentiment
UAS Natural Language Processing - Sentiment Analysis Naive Bayes

Ujian Akhir Semester
Mata Kuliah: Natural Language Processing

Repository ini berisi implementasi sederhana algoritma Naive Bayes
untuk klasifikasi sentimen teks (positif, netral, negatif).

Dataset yang digunakan berasal dari dokumen dataset_sentimen.pdf
yang diberikan oleh dosen pada UAS.

Bahasa pemrograman yang digunakan adalah Python.
Model dibangun tanpa library machine learning eksternal.
